**Why:** 
Áp dụng đối với các dự án xây dựng AI Assistant cho giáo dục, đào tạo nội bộ, nơi agent cần đóng vai "Gia sư" có khả năng hướng dẫn thích ứng (adaptive learning workflow), chia nhỏ bài toán thay vì chỉ nhả output trực tiếp.

**How to apply:** 
Hãy di chuyển vào repo `d:\code\DeepTutor`, tham chiếu mã nguồn để mổ xẻ kiến trúc RAG, các luồng phân tách prompt (Prompt engineering module) và tích hợp các tư duy sư phạm này vào Agent Prompt Template của Antigravity khi phát triển nền tảng Elearning.
