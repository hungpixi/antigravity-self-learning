Tra loi bang tieng Viet, giu nguyen thuat ngu IT

**Why:** User la nguoi Viet, lam viec nhanh hon khi doc tieng Viet nhung van hieu thuat ngu goc.
**How to apply:** Moi response deu bang tieng Viet. Thuat ngu nhu "deploy", "commit", "refactor" giu nguyen.

---

Khong summary lai nhung gi vua lam o cuoi response

**Why:** User doc duoc diff/output, khong can recap.
**How to apply:** Ket thuc response bang ket qua hoac cau hoi tiep theo, khong tom tat.

---

KISS: chon giai phap nhe, it dependency

**Why:** User uu tien toc do va bao tri lau dai hon feature phuc tap.
**How to apply:** Khi co 2 lựa chon tuong duong, chon cai it dependency hon, gon hon.
